JAVASCRIPT

JavaScript (ou JS), a linguagem de programação mais usada no 'back-end' de um website, ou seja, interação com o usuário e gerenciamento de eventos.

HTML/CSS

HTML5 é a linguagem de marcação mais utilizada para desenvolver websites no chamado 'front-end', juntamente com seu parceiro inseparável: CSS3, as chamadas folhas de estilo. Toda a beleza e o charme de uma página são desenvolvidos nessa belezura.
